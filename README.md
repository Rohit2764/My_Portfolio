## 🧠 Features

- 📁 **Projects Gallery** with links to GitHub
- 🧾 **Resume Download**

- **React** with TypeScript

- **Vite** as the build tool
- **Shadcn UI** components
- **EmailJS** or similar for contact form (optional)
- **React Icons** for tech icons and social links
